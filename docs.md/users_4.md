## User Stories

1. As a student, I want to create a post so that I can sell items or share housing information.

2. As a student, I want to log in using my school email so that only students can access the platform.

3. As a student, I want to view my posts so that I can track their approval status.

4. As a student, I want to search listings so that I can quickly find items or housing.

5. As a student, I want to report suspicious posts so that moderators can review them.

6. As a moderator, I want to approve posts so that valid posts become visible to students.

7. As a moderator, I want to reject inappropriate posts so that the platform remains safe.

8. As an admin, I want to manage users so that the platform operates smoothly.

9. As a student, I want to filter posts by category so that I can easily browse listings.
